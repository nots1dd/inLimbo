inLimbo - TUI Music Player

A TUI-based music player for seamless audio playback and control with efficient metadata extraction and viewing

About

inLimbo is a terminal-based music player that provides an immersive experience for managing your music library. Built using C++, it allows you to control music playback, view metadata, and organize your songs all from within a sleek text-based interface.

Features

  • Seamless audio playback and control
  • Efficient metadata extraction and viewing
  • Intuitive keyboard shortcuts for ease of use
  • Support for local and streaming music sources
  • Customizable user interface with theming options
  • Lightweight and responsive performance
