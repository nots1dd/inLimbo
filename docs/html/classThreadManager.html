void ThreadManager::unlockPlayMutex ( ThreadState & thread_state)
inline

Unlocks the play mutex for the provided thread state.

This function releases the lock on the play mutex that was previously acquired using lockPlayMutex. The play mutex will be unlocked after this function is called.

Parameters
thread_stateA reference to the ThreadState object whose mutex is to be unlocked.
Note
This function should be used in conjunction with lockPlayMutex to ensure proper synchronization.
