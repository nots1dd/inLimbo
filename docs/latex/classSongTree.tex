\doxysubsection{Detailed Description}
Represents a hierarchical tree structure to store songs by artist, album, disc number, and track number. 

The {\ttfamily \doxylink{classSongTree}{Song\+Tree}} class organizes songs in a nested map structure, allowing for efficient storage and retrieval of songs based on artist, album, disc, and track numbers. It supports the addition, display, and serialization of songs, and provides methods for querying songs by specific criteria such as artist, album, and genre.

Songs are added to the tree via the {\ttfamily add\+Song} method, and the tree can be serialized and deserialized to a file using the Cereal library.
