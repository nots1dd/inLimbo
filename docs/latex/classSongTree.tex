\doxysubsection{Detailed Description}
Represents a hierarchical tree structure to store songs by artist, album, disc number, and track number. 

The \doxylink{classSongTree}{Song\+Tree} class organizes songs in a map structure where the keys are artist names, album titles, disc numbers, and track numbers. The tree structure allows for easy retrieval of songs by artist, album, and track number. Songs can be added to the tree, displayed, and serialized or deserialized from a file.
