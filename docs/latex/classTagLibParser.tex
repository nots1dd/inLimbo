\doxysubsection{Detailed Description}
A class for parsing metadata from audio files. 

The {\ttfamily \doxylink{classTagLibParser}{Tag\+Lib\+Parser}} class uses the Tag\+Lib library to extract metadata from audio files. It supports parsing tags, including title, artist, album, genre, year, and track information. Additionally, it can parse metadata from files using inode information.
