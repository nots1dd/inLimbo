\doxysubsection{Detailed Description}
A header file for the \doxylink{classTagLibParser}{Tag\+Lib\+Parser} class and \doxylink{structMetadata}{Metadata} structure, used to parse metadata from audio files. 

This file defines a {\ttfamily \doxylink{structMetadata}{Metadata}} structure for storing song information such as title, artist, album, and more. It also defines the {\ttfamily \doxylink{classTagLibParser}{Tag\+Lib\+Parser}} class for reading and extracting metadata from audio files using the Tag\+Lib library. In addition, the file provides utility functions for printing metadata, handling error messages, and extracting thumbnails from audio files.

{\bfseries{Note}}\+: In the Emscripten build, dummy functions are used because Tag\+Lib must be compiled using Emscripten to function correctly in a web environment.
