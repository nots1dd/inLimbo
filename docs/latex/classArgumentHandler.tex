\doxysubsection{Detailed Description}
Handles processing of command-\/line arguments and executes corresponding actions. 

The {\ttfamily \doxylink{classArgumentHandler}{Argument\+Handler}} class is responsible for parsing command-\/line arguments, executing predefined actions for valid flags, and managing application paths. It also includes utilities for colorful console output.
