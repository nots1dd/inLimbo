\doxysubsection{Detailed Description}
Represents a song with associated metadata and inode. 

The {\ttfamily \doxylink{structSong}{Song}} struct encapsulates the metadata and inode of a song, including details such as title, artist, album, genre, and track number. It provides methods to initialize, serialize, and deserialize song data.
