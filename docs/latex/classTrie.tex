\doxysubsection{Detailed Description}
Implements the \doxylink{classTrie}{Trie} (Prefix Tree) data structure. 

This class provides methods to insert artist names into the \doxylink{classTrie}{Trie} and search for all names that start with a given prefix. It stores a list of indices for each node, making it efficient to retrieve all matching names for a prefix.
