\doxysubsection{Detailed Description}
Represents a single node in the \doxylink{classTrie}{Trie}. 

A \doxylink{classTrieNode}{Trie\+Node} stores children nodes for each possible character, a list of indices representing the positions of words in the original dataset, and a flag indicating if the node is the end of a valid word.

\begin{DoxyNote}{Note}
The {\ttfamily children} unordered\+\_\+map stores the next character in the string as a key and the corresponding child node as the value. 
\end{DoxyNote}
