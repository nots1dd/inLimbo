\doxysubsection{Detailed Description}
A custom scroller component that synchronizes with an external index and applies a custom cursor background color. 

The {\ttfamily \doxylink{classftxui_1_1ScrollerBase}{Scroller\+Base}} component disables internal navigation and allows the scroll state to be controlled externally using an external index ({\ttfamily external\+\_\+selected\+\_\+}). It also supports custom styling for the focused element using a background color ({\ttfamily cursor\+\_\+bg\+\_\+}).
