\doxysubsection{Detailed Description}
Represent an event. It can be key press event, a terminal resize, or more ... 

For example\+:
\begin{DoxyItemize}
\item Printable character can be created using \doxylink{structftxui_1_1Event_a76bb32507ef5b345b33a5e5858fd4f22}{Event\+::\+Character}(\textquotesingle{}a\textquotesingle{}).
\item Some special are predefined, like Event\+::\+Arrow\+Left.
\item One can find arbitrary code for special Events using\+: ./example/util/print\+\_\+key\+\_\+press For instance, CTLR+A maps to \doxylink{structftxui_1_1Event_a96a1fca3c470b925314dc68bb4ff69fa}{Event\+::\+Special}(\{1\});
\end{DoxyItemize}
